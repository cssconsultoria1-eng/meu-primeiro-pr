# meu-primeiro-pr

Repositório criado para praticar o fluxo de contribuição com Pull Requests no GitHub.

## O que é um Pull Request?

Um Pull Request (PR) é uma forma de propor mudanças em um repositório. Você cria um branch, faz alterações e abre um PR para que as mudanças sejam revisadas antes de serem incorporadas ao branch principal.

## Como contribuir

1. Faça um fork ou clone do repositório
2. Crie um branch para sua alteração: `git checkout -b minha-melhoria`
3. Faça suas alterações e commite: `git commit -m "Descrição da melhoria"`
4. Envie o branch: `git push origin minha-melhoria`
5. Abra um Pull Request no GitHub
